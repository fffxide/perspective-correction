# 斜め撮り → 正面化

斜めから撮影された画像を、正面から撮ったように補正するブラウザ完結のWebアプリです。
画像処理はすべてクライアントサイド(JavaScript)で行われ、サーバーには一切送信されません。

## 公開URL

https://fffxide.github.io/perspective-correction/

## 使い方

1. 画像をアップロードすると、四隅付近に4点が自動配置されます
2. 各点をドラッグして、補正したい四角形の角に合わせます（左上 → 右上 → 右下 → 左下 の時計回り）
3. 「補正実行」ボタンで結果を更新、または「リアルタイムプレビュー」をオンにすると点を動かすたびに自動更新
4. 「結果をダウンロード」で保存

## 操作

- 点をクリックで選択 → 矢印キーで1px、Shift+矢印で10px動かして微調整
- Ctrl+Z または「元に戻す」で直前のドラッグを取り消し
